## ğŸš€ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/travhall/thd-portfolio.git
   ```

2. **Install dependencies**
   ```bash
   cd thd-portfolio
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Start production server**
   ```bash
   npm run start
   ```
